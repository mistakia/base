# Create Guideline

## Guidelines

- Guidelines MUST be named like `guideline_name.md`.
- Guidelines MUST use RFC 2119 language (MUST, SHOULD, MAY, etc.).

## Observations

- [governance] Clear naming conventions improve discoverability #naming
- [standard] RFC 2119 language provides clarity on requirement levels #clarity
- [principle] Guidelines establish clear boundaries for activities #structure
