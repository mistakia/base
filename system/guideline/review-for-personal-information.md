Personal information includes anything that can be used to discover a specific individual.

- Content that can be used to identify a specific location
- be mindful of information that can be used to significantly narrow the pool of possible individuals

## Allowable Information

The following are NOT considered personal information for privacy purposes:

- The username "trashman" in file paths and system references
