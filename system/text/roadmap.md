# Roadmap

This document outlines planned features and improvements for the system.

## Notes

- Add support for reranking in retrieval
- Create an entity deduplication thread/workflow
- Layered Knowledge Graph
  - Raw Graph (non-lossy raw data)
  - Entity Graph (knowledge base)
  - Semantic Graph (derived from entity graph)
  - Community Graph (highest level graph representing clusters of strongly connected entities)
